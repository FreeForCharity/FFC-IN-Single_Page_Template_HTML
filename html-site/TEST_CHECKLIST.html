Free For Charity - HTML Site Validation

Manual Testing Checklist

1. HTML Structure ✓

  • Valid HTML5 doctype
  • Proper head section with meta tags
  • Semantic HTML elements (header, nav, main, section, footer)
  • All tags properly closed

2. Navigation ✓

  • Desktop navigation links work
  • Mobile menu toggle works
  • Smooth scrolling to sections
  • Active section highlighting

3. Sections ✓

  • Hero section renders correctly
  • Mission section with video
  • Results section with animated numbers
  • Volunteer section
  • Events section with iframe
  • Donate section with iframe
  • Programs section
  • FAQ accordion works
  • Team section

4. Interactive Features ✓

  • Cookie consent banner shows on first visit
  • Cookie preferences modal works
  • FAQ accordion expands/collapses
  • All links navigate correctly
  • External links open in new tab

5. Responsive Design

  • Test on mobile (375px width)
  • Test on tablet (768px width)
  • Test on desktop (1280px+ width)
  • Images scale properly
  • Text is readable on all devices

6. Assets

  • All images load
  • All SVGs load
  • Favicon displays
  • Video poster image displays

7. SEO & Accessibility

  • Title tag present
  • Meta description present
  • Open Graph tags present
  • Alt text on images
  • ARIA labels on interactive elements
  • Proper heading hierarchy (h1 -> h2 -> h3)

Browser Testing

  • ✓ Chrome/Edge
  • ✓ Firefox
  • ✓ Safari (desktop)
  • ✓ Mobile Safari (iOS)
  • ✓ Chrome Mobile (Android)

Performance Metrics (Target)

  • First Contentful Paint: < 1.5s
  • Largest Contentful Paint: < 2.5s
  • Total Blocking Time: < 300ms
  • Cumulative Layout Shift: < 0.1
  • Speed Index: < 3.5s

Validation Tools

  • W3C HTML Validator
  • W3C CSS Validator
  • WAVE Accessibility Checker
  • PageSpeed Insights

Note: This is a manual testing checklist. Run through each item to verify the HTML site is working correctly.
